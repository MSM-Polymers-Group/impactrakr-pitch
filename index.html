  • Imutabilidade absoluta de eventosAbsolute event immutability
    Sem edição. Correção é evento novo, nunca sobreposição. A trilha de evidência só cresce — nunca é reescrita.No edits. Corrections are new events, never overwrites. The evidence trail only grows — never rewritten.

  • ADRs públicos para toda decisão arquiteturalPublic ADRs for every architectural decision
